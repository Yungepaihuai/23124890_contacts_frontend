# Address Book Management System-Front End
23124890_contacts_frontend

## Project Introduction
    The front-end of the address book management system based on 
Vue.js adopts the front-end and back-end separation architecture.

## Technology stack
  HTML5
  CSS3
  JavaScript (ES6+)
  Vue.js 2.x

## Functional
  Add contacts
  Edit contact  
  Delete contact
  View contact list
  Responsive design

## Operating instructions
  Method 1：Open http://127.0.0.1:5500/scr/index.html Method 2: Install Live Server plug-in in VS Code.Right click index.html choose "Open with Live Server"
